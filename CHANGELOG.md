# Changelog

## [0.24.0] - 2026-04-19

- 工具栏支持拖拽调整宽度（328-800px，localStorage 持久化）
- 模板套装系统重构为 v2 扁平模型，修复 onunload 回写磁盘导致删除无效的问题
- 移除 Entrepreneur Journey 模板套装
- 引用块样式根据当前主题自适应（typora-newsprint / wabi-sabi / default）
- 默认主题色从紫色调整为中性灰蓝

## [0.23.4] - 2026-04-02

- 修复表格上传按钮点击无效和截图不完整
- 修复插件 toggle / 配置变更后渲染丢失
- 修复 Shadow DOM 下主题色读取失败
- 修复表格表头颜色与主题不协调
- 优化模板套装卡片布局

## [0.23.0] - 2026-01-15

- 新增 PDF 导出选项
- 修复 Shadow DOM 模式下截图功能失效

## [0.22.3] - 2025-12-16

- 开发工具升级迁移

- 表格支持复制为图片、上传为图片功能

## [0.22.1] - 2025-12-10

- 复制图片时自动缩放溢出的代码块，确保完整显示
- 新增图片预览模式，可预览截图效果
- 工具栏支持左右位置切换
- 设置界面全新温暖学术设计风格
- 修复 Obsidian 中头像菜单无法点击的问题
