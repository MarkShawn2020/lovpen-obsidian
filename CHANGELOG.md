# Changelog

## [0.25.4] - 2026-05-09

- 支持 Obsidian wiki link 格式 `[[Page]]`、`[[Page|Alias]]`、`[[Page#Heading]]` 的格式化
- 外部 URL 形式的 wiki link 会复用现有链接脚注处理链路

## [0.25.3] - 2026-05-09

- 修复 BRAT 安装后点击模板套装提示“主题资源未下载”的问题
- 将内置主题与代码高亮资源随插件主文件打包，首次加载时自动补齐本地资源目录

## [0.25.2] - 2026-05-09

- 修复 BRAT 安装后模板套装列表为空的问题
- 将内置模板套装和 HTML 模板打包进插件主文件，缺失资源时自动补齐
- 为 BRAT 三件套安装场景补齐 `template-kits.json` 与 `assets/templates/*.html`

## [0.25.1] - 2026-05-09

- 支持通过 BRAT 从 GitHub Release 安装 LovPen 插件
- 发布资产新增 `main.js`、`manifest.json`、`styles.css` 和 `assets.zip`
- 修复插件资源路径硬编码，兼容 BRAT 的 `lovpen` 插件目录

## [0.25.0] - 2026-04-19

- 封面提示词升级为两步推理（keyIdea 提炼文章主旨 → visualMetaphor 视觉隐喻 → positivePrompt），取样范围扩大到头/中/尾三段拼接
- 封面生成支持参考图（仅 Nano Banana Pro 生效）：从已上传图库选择一张作为视觉参考
- 重构：七牛云上传逻辑从 Toolbar.tsx 抽取到独立 qiniuUpload 服务

## [0.24.0] - 2026-04-19

- 工具栏支持拖拽调整宽度（328-800px，localStorage 持久化）
- 模板套装系统重构为 v2 扁平模型，修复 onunload 回写磁盘导致删除无效的问题
- 移除 Entrepreneur Journey 模板套装
- 引用块样式根据当前主题自适应（typora-newsprint / wabi-sabi / default）
- 默认主题色从紫色调整为中性灰蓝

## [0.23.4] - 2026-04-02

- 修复表格上传按钮点击无效和截图不完整
- 修复插件 toggle / 配置变更后渲染丢失
- 修复 Shadow DOM 下主题色读取失败
- 修复表格表头颜色与主题不协调
- 优化模板套装卡片布局

## [0.23.0] - 2026-01-15

- 新增 PDF 导出选项
- 修复 Shadow DOM 模式下截图功能失效

## [0.22.3] - 2025-12-16

- 开发工具升级迁移

## [0.22.2] - 2025-12-15

- 表格支持复制为图片、上传为图片功能

## [0.22.1] - 2025-12-10

- 复制图片时自动缩放溢出的代码块，确保完整显示
- 新增图片预览模式，可预览截图效果
- 工具栏支持左右位置切换
- 设置界面全新温暖学术设计风格
- 修复 Obsidian 中头像菜单无法点击的问题
